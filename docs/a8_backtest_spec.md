# A8統合バックテスト

## 正式な起動方法

```powershell
python backtest_a8.py
```

旧入口の `backtest_vwap.py` と `backtest_A_a7r2.py` は削除済み。正式入口は `backtest_a8.py` のみとする。

一度選択した監視銘柄ファイルは `.a8_settings.json` に記憶する。次回起動時にファイルが存在すれば自動入力し、存在しない、または設定を読み込めない場合はファイル選択画面を表示する。

## GUI設定

- 開始日、終了日
- 監視銘柄Markdownファイル
- 出力先フォルダ
- 25日乖離率の最低値、最高値
- VWAP維持確認あり／なし
- エントリー時刻
  - VWAP維持確認あり: 前日終値、11:00、14:00
  - VWAP維持確認なし: 11:00、14:00
- 直近3営業日の安値切り下げ除外回数: 0回、1回、2回、3回

開始日の初期値は終了日の40営業日前、終了日の初期値は直近の営業日とする。営業日の初期値計算は土日を除く平日基準で行う。

安値切り下げは、直近3営業日それぞれの安値を前営業日と比較して数える。0回は条件を無効化し、1～3回は実際の回数が選択値以上の場合に除外する。

## エントリー条件

次の条件をすべて満たす場合にエントリーする。

- 前日と判定時点の25日乖離率が設定範囲内
- 25日線が横ばい以上
- VWAP維持確認ありの場合、判定価格が累積VWAP以上
- VWAP維持確認なしの場合、VWAPとの位置関係にかかわらず指定時刻にエントリー
- 安値切り下げ回数が除外基準未満、または除外設定が0回

## 結果解析

5・10・20営業日の各期間について、次を集計する。

- 最大順行幅（MFE）中央値
- 最大逆行幅（MAE）中央値
- -3%逆行率
- +5%到達率

出力ファイル名は `backtest_a8_summary-*` と `backtest_a8_result-*` とする。
